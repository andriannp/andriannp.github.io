# Personal Portfolio - Coming Soon Page

Repositori ini berisi landing page **"Coming Soon"** yang modern, minimalis, dan elegan untuk website portfolio pribadi Anda. Didesain khusus untuk profil profesional seorang *Strategic Planning Specialist & Certified Marketer* dengan performa ringan dan animasi premium.

## Fitur Utama
1. **Glassmorphism**: Konten utama dibungkus dalam kartu semi-transparan ber-blur tinggi.
2. **Animated Gradient Mesh**: Latar belakang dinamis dengan gelembung warna (*mesh blobs*) yang bergerak lembut secara otomatis.
3. **Cursor Parallax**: Efek interaktif di desktop di mana latar belakang bergeser halus merespon pergerakan mouse.
4. **Entrance Animations**: Transisi masuk elemen secara berurutan (*staggered animations*) saat halaman dibuka.
5. **Dark Mode Default**: Warna gelap mendalam yang nyaman di mata dan terlihat premium.

## Langkah Kustomisasi Foto Profil Anda

Saat ini halaman memuat file gambar dari lokasi `assets/images/profile.jpg`. Untuk menggantinya dengan foto formal Anda:
1. Siapkan file foto Anda (misalnya yang bernama `Andrian Nur Prabawa Jas Formal.png`).
2. Masukkan file tersebut ke dalam folder `assets/images/` di repositori ini.
3. Ubah nama file (*rename*) dari `Andrian Nur Prabawa Jas Formal.png` menjadi **`profile.jpg`** (menggantikan file placeholder yang lama).
4. Commit dan push perubahan tersebut ke GitHub. Foto Anda akan langsung terupdate di halaman website!

## Cara Mendeploy ke GitHub Pages

Karena website ini dibangun hanya menggunakan HTML, CSS, dan JS murni (tanpa framework berat), proses deploy-nya sangat mudah dan gratis:

1. **Push Berkas Anda**:
   Pastikan semua berkas (`index.html`, `style.css`, `script.js`, dan folder `assets/`) telah di-commit dan di-push ke branch utama (`main`) di repositori GitHub Anda: `github.com/andriannp/andriannp.github.io`.

2. **Aktifkan GitHub Pages**:
   - Masuk ke repositori Anda di GitHub.
   - Klik tab **Settings** di menu atas.
   - Di menu sebelah kiri, cari bagian **Code and automation**, lalu klik **Pages**.
   - Di bawah bagian **Build and deployment**:
     - **Source**: Pilih *Deploy from a branch*.
     - **Branch**: Pilih `main` dan folder `/ (root)`.
   - Klik tombol **Save**.

3. **Verifikasi**:
   GitHub akan memicu *deployment* otomatis. Dalam waktu 1-2 menit, website Anda akan aktif dan dapat diakses di alamat domain kustom Anda: **`https://andrian.id`**.
